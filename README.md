# TryHackMe-CN

> [!CAUTION]
> 由于本项目涉及到 TryHackMe 平台独有的内容和教学内容，本项目自身承诺  **CC0-1.0 license** 许可证并不将本项目的内容用于商业行为，并且本项目也不提倡将项目中的文件用于其他用途，本项目本质是为中国大陆地区的网络安全爱好者，提供一个更加方便的途径来学习 TryHackMe 平台上面的内容。

## 加入本项目

TryHackMyOffsecBox 欢迎**中国大陆**的 TryHackMe 活跃用户以及网络安全爱好者共同加入本项目，本项目核心任务分为两块

1. 将 **TryHackMe** 平台上的内容整理为格式完整的 **Markdown** 文档
2. 使用 **Crowdin** 平台，将原始的 **英语** 内容翻译为 **简体中文** 版本

本项目分为多个阶段，现有执行计划如下

- [ ] 对 TryHackMe 平台上的 `Modules` 内容进行搬运并翻译
- [ ] 对 TryHackMe 平台上的 `Networks` 内容进行搬运并翻译
- [ ] 对 TryHackMe 平台上的 `Roadmap` 内容进行搬运与整理

## 如何实时运行本项目

运行英语版本

```shell
npm run start -- --locale en
```

运行简体中文版本

```shell
npm run start -- --locale zh
```
