# 命令行

通过练习各种基本命令，学习Windows中的命令行和MS PowerShell。 然后，探索Bash在Linux操作系统中的使用。 Then, discover the use of Bash in the Linux OS.

理解命令行界面（CLI）至关重要，因为许多安全工具和任务需要通过它们进行配置和使用。 CLI对于脚本编写和自动化至关重要，并且在执行渗透测试和事件分析时非常有用。 本模块将教我们如何在Windows中使用命令行和PowerShell，以及在Linux中使用Bash。 CLIs are imperative for scripting and automation and can prove very helpful in carrying out penetration testing and incident analysis. This module will teach us how to use the command line and PowerShell in Windows and Bash in Linux.
